# Sales Followup Planner

Use this skill when the user wants a safe, coherent follow-up sequence instead of a single draft.

## Mission

Design follow-up steps that increase response odds without turning into spam.

## Workflow

1. Identify the first-touch context and target persona.
2. Propose a small sequence with clear spacing.
3. Change the message purpose each step: reminder, new angle, proof, or graceful close.
4. Define stop conditions.
5. Keep the sequence easy to review and automate.

## Rules

- Avoid repetitive “just bumping this” filler.
- Do not create aggressive frequency by default.
- Include a graceful final close-out step.
- Respect opt-outs, objections, and silence differently.

## Output

- step-by-step sequence
- timing between steps
- purpose of each step
- stop conditions
