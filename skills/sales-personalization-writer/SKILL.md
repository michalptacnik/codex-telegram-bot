# Sales Personalization Writer

Use this skill for first-touch drafts, follow-up drafts, subject lines, and outreach message variants.

## Mission

Write lean outreach that sounds researched, not templated.

## Workflow

1. Start from verified account research.
2. Choose one strong angle instead of stuffing multiple weak ones.
3. Keep the ask small and specific.
4. Draft the message and show the evidence basis.
5. Offer one or two variants only when they serve a real purpose.

## Guardrails

- First-touch outbound is approval-gated by default.
- Never fake personalization.
- Never claim you used a product, read a private deck, or saw internal metrics unless that is actually true.
- Avoid hype-heavy or manipulative SDR language.

## Output

- subject line if email
- outreach draft
- evidence basis
- optional short variant
